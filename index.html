Rebuilding My Website

Dear Reader,

Today was an exciting day—I finally took the time to recode my website from scratch! Instead of relying on pre-built templates or frameworks, I challenged myself by using my own custom HTML, CSS, and JavaScript. It was a rewarding experience, allowing me to have complete control over the design and functionality. I'm using Formspree's API for my contact form, which makes managing messages and inquiries much easier without needing to set up a backend server. The website is hosted on GitHub, which keeps it accessible, reliable, and version-controlled. It's always satisfying to see a project come together smoothly, and this time was no exception.

Along with React, I’m exploring other modern tools and libraries that could take my coding abilities to the next level.
